# Employee Management System

A Java-based web application built using Servlets, JSP, and JDBC.

## 🔹 Features

- Admin can add, update, and delete employee records
- Employees can view their own details
- Authentication and session management implemented
- Password encryption using BCrypt

## 🔹 Frontend

- Integrated UI (restaurant-themed) for better user experience
- Includes pages like Home, Menu, Contact, About

## 🔹 Tech Stack

- Java (Servlets, JSP)
- JDBC
- MySQL
- HTML, CSS, BootStrap

## 🔹 Note

The UI is designed for demonstration purposes, 
while the core functionality focuses on employee management.
